# 💸 Мои финансы

Личный дашборд трат по выписке Т-Банка («Справка о движении средств»)
за 08.01.2026 – 08.07.2026. Статический сайт без бэкенда — хостится на
GitHub Pages, открывается с телефона.

## Что умеет

- помесячный график трат по категориям, KPI-плитки, поиск и фильтры
  (период, тип операций, категория);
- автокатегоризация по правилам-подстрокам (`scripts/categorize.py`);
- свои категории: «＋ Своя категория» или из карточки операции — имя,
  эмодзи, цвет, правила. Хранятся в localStorage браузера, как и ручные
  правки категорий отдельных операций;
- выгрузка отфильтрованных операций в CSV (`операции.csv`), плюс готовая
  таблица `transactions.csv` в корне репозитория.

## Шифрование

Репозиторий публичный (условие бесплатного GitHub Pages), но операции в
открытом виде в него не попадают: в git лежит только `data.enc.js` —
AES-256-GCM, ключ из пароля через PBKDF2 (600 000 итераций SHA-256).
Сайт при открытии спрашивает пароль и расшифровывает данные прямо в
браузере; пароль можно запомнить на устройстве или передать один раз в
ссылке как `…/#p=пароль` (из адресной строки он сразу затирается).

## Файлы

| Файл | Что это |
|---|---|
| `index.html`, `style.css`, `app.js` | сайт |
| `data.enc.js` | зашифрованные операции + базовые категории |
| `scripts/parse_statement.py` | парсер PDF-выписки (pdfplumber), сверяет суммы с итогами банка |
| `scripts/categorize.py` | категории, правила, санитизация → `data.js` + `transactions.csv` (локально, в git не попадают) |
| `scripts/encrypt_data.mjs` | `data.js` → `data.enc.js` |

## Обновить данные по новой выписке (или сменить пароль)

```bash
pip install pdfplumber
python3 scripts/parse_statement.py /tmp/transactions.json   # путь к PDF внутри скрипта
python3 scripts/categorize.py /tmp/transactions.json data.js
node scripts/encrypt_data.mjs data.js data.enc.js "новый-пароль"
git add data.enc.js && git commit -m "Обновление данных" && git push
```

Сам PDF, `data.js` и `transactions.csv` в репозиторий не кладутся
(`.gitignore`). Номера телефонов и счетов в описаниях замаскированы ещё
на этапе генерации.
